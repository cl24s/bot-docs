Üdvözöl a Trading Bot Dokumentáció

Bevezetés

Ez egy automatizált trading bot rendszer dokumentációja, amely AI-alapú döntéshozatalt használ a cryptocurrency kereskedéshez.

v1.0.0

Főbb funkciók

  • AI-alapú kereskedési döntések
  • Automatikus pozíció nyitás és zárás
  • Take Profit és Stop Loss kezelés
  • Valós idejű monitoring dashboard
  • Részletes logging és analytics

Telepítés

A rendszer telepítéséhez kövesd az alábbi lépéseket:

Előfeltételek

Python 3.8+
pip
Binance API kulcsok

Telepítési lépések

  1. Klónozd a repository-t
  2. Telepítsd a függőségeket: pip install -r requirements.txt
  3. Konfiguráld az API kulcsokat
  4. Indítsd el a dashboard-ot: python cla/app.py
