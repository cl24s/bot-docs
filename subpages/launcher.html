Launcher - Egységes Indító

A Launcher egy egységes indító script, amely különböző bot utility-ket futtat egyetlen parancsból. Alapértelmezetten a CLA (Command Line Application) web interfészt indítja.

Áttekintés

python launcher.py           # CLA indítása (alapértelmezett)
python launcher.py -i        # Log importer futtatása
python launcher.py -u        # Database update futtatása

Főbb funkciók

  • CLA web interface indítása
  • Log import automatizálás
  • Database frissítés
  • Market context generálás
  • Kombinált műveletek

Főbb Műveletek

1. CLA Indítása (Alapértelmezett)

Indítja a Flask-alapú web dashboard-ot bot kezeléshez.

python launcher.py
  • Automatikusan a cla/ könyvtárba vált
  • Futtatja az app.py Flask alkalmazást
  • Web interface elérhető: http://localhost:5000

2. Log Importer (-i, --import-logs)

Importálja a szegmentált log fájlokat az adatbázisba.

python launcher.py -i

# Opciók:
--status              # Import státusz megjelenítése
--logs-dir logs      # Log könyvtár (alapértelmezett: logs)
--grace-minutes 10  # Grace period percben

3. Database Update (-u, --update)

Frissíti az adatbázist friss piaci adatokkal.

python launcher.py -u

# Opciók:
--download-only       # Csak letöltés
--import-only         # Csak import
--symbol ETHUSDT     # Szimbólum (alapértelmezett: ETHUSDT)
--days 7              # Napok száma (alapértelmezett: 7)

4. Teljes Frissítés (-a, --all)

Futtatja a log import-ot ÉS a database update-et egymás után.

python launcher.py -a

# Lépések:
1. Log importer futtatása
2. Database update futtatása
3. Sikeres befejezés jelentése

Használati Példák

Példa 1: CLA indítása

python launcher.py

Elindítja a web dashboard-ot a böngészőben.

Példa 2: Log import státusz ellenőrzése

python launcher.py -i --status

Megjeleníti az import státuszt import futtatása nélkül.

Példa 3: Database frissítés csak letöltéssel

python launcher.py -u --download-only --symbol BTCUSDT --days 14

Letölti a BTCUSDT adatokat 14 napra, de nem importálja.

Példa 4: Teljes frissítés egyedi beállításokkal

python launcher.py -a --logs-dir custom_logs --symbol SOLUSDT

Futtatja a teljes frissítést egyedi log könyvtárral és szimbólummal.

Munkafolyamat

Launcher működése

  1. Argumentum parsing: Feldolgozza a parancssori argumentumokat
  2. Akció meghatározás: Eldönti, melyik műveletet kell végrehajtani
  3. Végrehajtás: Futtatja a megfelelő modult
  4. Eredmény jelentés: Visszajelzés a művelet sikeréről/kudarcáról

CLA Launch folyamat

1. Váltás a cla/ könyvtárba
2. Flask app indítása (app.py)
3. Web interface elérhetővé válik
4. Ctrl+C leállítja az alkalmazást

Teljes frissítés folyamat

STEP 1: Log Importer
- Szegmentált log fájlok beolvasása
- Adatbázisba importálás
- Duplikátum ellenőrzés

STEP 2: Database Update
- Piaci adatok letöltése Binance API-ról
- Adatok feldolgozása
- Adatbázis frissítése

RESULT:
✅ Teljes frissítés sikeres!

Hibakezelés

Gyakori hibák

  • CLA launch hiba: Ellenőrizd, hogy a cla/app.py létezik
  • Import hiba: Győződj meg róla, hogy a log könyvtár létezik
  • Database hiba: Ellenőrizd az adatbázis kapcsolatot
  • Module not found: Telepítsd a szükséges függőségeket

Debug módok

Minden modul részletes hibaüzeneteket ad vissza:

❌ Error running log importer: [részletes hibaüzenet]
❌ Error running database update: [részletes hibaüzenet]
