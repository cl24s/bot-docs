A Trading Bot parancssori alkalmazás, amely Python argumentumokkal konfigurálható. Az alábbi útmutató részletezi az összes elérhető opciót és használati módot.

Áttekintés

A bot a main.py fájl futtatásával indítható, különböző argumentumokkal testreszabva a működését.

python main.py -s strategy_name --symbol ETHUSDT

Alapvető működés

  • Betölti a megadott stratégiát
  • Lekéri a piaci adatokat a Binance API-ról
  • AI döntést kér az OpenAI API-tól
  • Végrehajtja a kereskedést (opcionális)
  • Naplózza az összes lépést

Parancssori Argumentumok

Az összes elérhető argumentum részletes leírása:

-s, --strategy

--strategy basic_scalp

A használni kívánt stratégia neve. Alapértelmezett: basic_scalp

Elérhető stratégiák a strategies/ könyvtárban találhatók.

--symbol

--symbol ETHUSDT

A kereskedési szimbólum (pl. BTCUSDT, ETHUSDT). Alapértelmezett: ETHUSDT

--test_mode

--test_mode

Binance testnet mód engedélyezése. Éles kereskedés helyett teszt környezetben fut.

-po, --prompt_only

-po  # vagy  --prompt_only

Csak a prompt generálása, AI hívás és kereskedés nélkül. Hasznos prompt teszteléshez és debug-hoz.

-ao, --ai_only

-ao  # vagy  --ai_only

Csak AI hívás, kereskedés végrehajtása nélkül. Az AI döntést megjeleníti, de nem hajt végre trade-et.

-lt, --loop_time

--loop_time 30

Ciklus időköz másodpercben. 0 = egyszeri futtatás. Alapértelmezett: 0

Ha értéke > 0, a bot folyamatosan fut a megadott időközönként.

-ac, --account

--account main

Fiók kiválasztása (main, sub1, sub2, sub3). Alapértelmezett: main

Több Binance fiók kezelésére szolgál.

-c, --capital

--capital 1000

Kereskedési tőke USDT-ben. 0 = fiók egyenleg használata. Alapértelmezett: 0

--log_file

--log_file custom_log.log

Egyedi log fájl név. Dashboard által használt opció.

Futtatási Módok

1. Prompt Only Mode

Csak a prompt generálása, hasznos fejlesztéshez és teszteléshez.

python main.py -s new_scalp_intra_strategy_4 --symbol BTCUSDT -po

Kimenet: Generált prompt fájlok a logs/ könyvtárban

2. AI Only Mode

AI döntés lekérése, de kereskedés végrehajtása nélkül.

python main.py -s basic_scalp --symbol ETHUSDT -ao

Kimenet: AI döntés JSON formátumban a konzolra és log fájlba

3. Full Trading Mode

Teljes kereskedési mód: prompt + AI + trade végrehajtás.

python main.py -s new_scalp_intra_strategy_4 --symbol SOLUSDT

Kimenet: Teljes kereskedési folyamat végrehajtása és naplózása

4. Loop Mode

Folyamatos futtatás megadott időközönként.

python main.py -s basic_scalp --symbol ETHUSDT -lt 60

A bot 60 másodpercenként újra futtatja a kereskedési logikát.

Használati Példák

Példa 1: Teszt futtatás AI-val, kereskedés nélkül

python main.py \
  -s new_scalp_intra_strategy_4 \
  --symbol BTCUSDT \
  -ao

Példa 2: Éles kereskedés 30 másodperces ciklussal

python main.py \
  -s basic_scalp \
  --symbol ETHUSDT \
  -lt 30 \
  --capital 500

Példa 3: Testnet mód prompt ellenőrzéssel

python main.py \
  -s new_scalp_intra_strategy_4 \
  --symbol SOLUSDT \
  --test_mode \
  -po

Példa 4: Sub account használata loop módban

python main.py \
  -s basic_scalp \
  --symbol BNBUSDT \
  --account sub1 \
  -lt 45

Log fájlok

Minden futtatás log fájlokat generál:

  • log_stdout/ - Konzol kimenet log-ok
  • logs/ - Strukturált JSON log-ok stratégiánként
  • logs/_last_prompt_for_ai.txt - Utolsó AI prompt
  • logs/_last_prompt_pretty.txt - Formázott prompt
